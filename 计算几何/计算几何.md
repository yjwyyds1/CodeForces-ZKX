# 计算几何

## 向量点积（内积）几何意义

### 向量的夹角关系
点积可以衡量两个向量之间的夹角关系。对于两个非零向量 **a** 和 **b**，它们的点积定义为：

\[
\mathbf{a} \cdot \mathbf{b} = \|\mathbf{a}\| \|\mathbf{b}\| \cos \theta=\mathbf{a}_{x}\mathbf{b}_{x}+\mathbf{a}_{y}\mathbf{b}_{y}
\]

其中：
- \(\theta\) 是它们之间的夹角
- \(\|\mathbf{a}\|\) 和 \(\|\mathbf{b}\|\) 分别是向量的模长

**结论：**
- 若 \(\mathbf{a} \cdot \mathbf{b} > 0\)，说明夹角 \(\theta\) 是锐角（\(0 \leq \theta < 90^\circ\)）
- 若 \(\mathbf{a} \cdot \mathbf{b} = 0\)，说明两向量正交（\(\theta = 90^\circ\)）
- 若 \(\mathbf{a} \cdot \mathbf{b} < 0\)，说明夹角 \(\theta\) 是钝角（\(90^\circ < \theta \leq 180^\circ\)）

### 投影长度
点积可以计算一个向量在另一个向量方向上的投影长度：

\[
\text{投影长度} = \|\mathbf{a}\| \cos \theta = \frac{\mathbf{a} \cdot \mathbf{b} }{\|\mathbf{b}\|}
\]




### 长度计算
一个向量的点积可以计算其模长的平方：

\[
\mathbf{a} \cdot \mathbf{a} = \|\mathbf{a}\|^2
\]

---
```c++
ld dot(point a,point b){//点积
    return a.x*b.x+a.y*b.y;
}
```

## 向量叉积（外积）

对于两个三维向量 **a** = (a₁, a₂, a₃) 和 **b** = (b₁, b₂, b₃)，其叉积定义为：

\[
\mathbf{a} \times \mathbf{b} = 
\begin{vmatrix}
\mathbf{i} & \mathbf{j} & \mathbf{k} \\
a_1 & a_2 & a_3 \\
b_1 & b_2 & b_3 \\
\end{vmatrix}
= (a_2b_3 - a_3b_2)\mathbf{i} - (a_1b_3 - a_3b_1)\mathbf{j} + (a_1b_2 - a_2b_1)\mathbf{k}
\]

___
对于两个二维向量 $\vec{a} = (a_{x},a_{y})$ 和 $\vec{b} = (b_{x},b_{y})$，其叉积定义为：

\[
\vec{a} \times \vec{b}=(a_{x}b_{y}-a_{y}b_{x})\hat{k} =\left | \vec{a}  \right | \left | \vec{b}  \right | \sin \theta\hat{k}
\]
### 几何意义

### 方向
- 叉积结果是一个**新的向量**，其方向垂直于原始两个向量所在的平面

### 大小
- 叉积向量的模长等于两个向量构成的平行四边形的面积：

\[
\|\mathbf{a} \times \mathbf{b}\| = \|\mathbf{a}\|\|\mathbf{b}\|\sin\theta
\]

- 由此可得三角形面积（平行四边形面积的一半）：

\[
\text{面积} = \frac{1}{2}\|\mathbf{a} \times \mathbf{b}\|  
\]

### 正交性判断
- 若两向量平行（夹角0°或180°），叉积为零向量

$$
\vec{a} \times \vec{a}= \vec{0} 
$$

$$
\vec{a} \times \vec{b}= \vec{0} \Longrightarrow \vec{a} // \vec{b}
$$

### 与点积的关系

### 标量三重积
\[
\mathbf{a} \cdot (\mathbf{b} \times \mathbf{c}) = \mathbf{b} \cdot (\mathbf{c} \times \mathbf{a}) = \mathbf{c} \cdot (\mathbf{a} \times \mathbf{b})
\]

表示三个向量构成的平行六面体的有向体积。

### 向量三重积
\[
\mathbf{a} \times (\mathbf{b} \times \mathbf{c}) = \mathbf{b}(\mathbf{a} \cdot \mathbf{c}) - \mathbf{c}(\mathbf{a} \cdot \mathbf{b})
\]


### 总结

与点积不同，叉积结果是一个向量而非标量，有**正负**方向之分，这使得它在描述方向性关系时具有独特优势
___

### 代码

```c++
ld crs(point a,point b){//叉积
    return a.x*b.y-a.y*b.x;
}
```

### 点线位置关系

$\hspace{15pt}$在二维平面上有两个不同的点 $A(x_A,y_A)$ 和 $B(x_B,y_B)$ 构成一条从 $A$ 到 $B$ 的直线 $AB$ ，判断点 $C$ 与其位置关系。具体地，你需要判断以下位置关系

- 点 $C$ 在向量 $\overrightarrow{AB}$ 左侧
- 点 $C$ 在向量 $\overrightarrow{AB}$ 右侧
- 点 $C$ 在线段 $AB$ 上（包括端点）
- 点 $C$ 在直线 $AB$ 上
```c++
#include <bits/stdc++.h>
const int N=2e5+10;
const int inf = 0x3f3f3f3f;
using namespace std;
using ull = unsigned long long int;
using ll = long long int;
using ld = long double;

struct point{
    ld x,y;
    point operator-(const point &a)const{
        return {x-a.x,y-a.y};
    }  
    ld operator^(const point &a)const{
        return x*a.y-y*a.x;
    }
};
int main() {
    ios::sync_with_stdio(false);
    cin.tie(0);
    int _;
    cin>>_;
    while(_--){
        point A,B,C;
        cin>>A.x>>A.y>>B.x>>B.y>>C.x>>C.y;
        point AB=(B-A);//向量AB
        point AC=(C-A);//向量AC
        if((AB^AC)>0){
            cout<<1<<'\n';//叉积大于零，逆时针，在左边
        }
        else if((AB^AC)<0){
            cout<<2<<'\n';//叉积小于零，顺时针，在右边
        }
        else{//叉积等于零就是在这条直线上
            if(min(A.x,B.x)<=C.x&&C.x<=max(A.x,B.x)&&min(A.y,B.y)<=C.y&&C.y<=max(A.y,B.y)){
                cout<<3<<'\n';
            }//C的横纵坐标在AB之间就是，C在AB线段上面
            else{
                cout<<4<<'\n';
            }
        }
    }
    return 0;
}
```

# 向量旋转公式及推导

## 1. 旋转理论基础

### 1.1 旋转矩阵定义
对于二维向量旋转：

- **逆时针旋转矩阵**：
  \[
  R(\theta) = \begin{pmatrix}
  \cos\theta & -\sin\theta \\
  \sin\theta & \cos\theta
  \end{pmatrix}
  \]

- **顺时针旋转矩阵**：
  \[
  R(-\theta) = \begin{pmatrix}
  \cos\theta & \sin\theta \\
  -\sin\theta & \cos\theta
  \end{pmatrix}
  \]

### 1.2 极坐标表示
设向量 \(\vec{a}\) 的极坐标形式：
\[
\vec{a} = \begin{pmatrix} r\cos\alpha \\ r\sin\alpha \end{pmatrix},\quad \text{其中}\ r = \|\vec{a}\| = \sqrt{x^2 + y^2}
\]

## 2. 旋转公式推导

### 2.1 逆时针旋转
旋转后的向量 \(\vec{a}'\)：
\[
\vec{a}' = R(\theta)\vec{a} = \begin{pmatrix}
x\cos\theta - y\sin\theta \\
x\sin\theta + y\cos\theta
\end{pmatrix}
\]

**极坐标验证**：
\[
\vec{a}' = r\begin{pmatrix} \cos(\alpha + \theta) \\ \sin(\alpha + \theta) \end{pmatrix}
\]

### 2.2 顺时针旋转
旋转后的向量 \(\vec{a}''\)：
\[
\vec{a}'' = R(-\theta)\vec{a} = \begin{pmatrix}
x\cos\theta + y\sin\theta \\
-x\sin\theta + y\cos\theta
\end{pmatrix}
\]

**极坐标验证**：
\[
\vec{a}'' = r\begin{pmatrix} \cos(\alpha - \theta) \\ \sin(\alpha - \theta) \end{pmatrix}
\]

## 3. 公式总结表
| 旋转方向 | 直角坐标公式 | 极坐标公式 |
|----------|--------------|------------|
| 逆时针 | \(\begin{pmatrix} x\cos\theta - y\sin\theta \\ x\sin\theta + y\cos\theta \end{pmatrix}\) | \(\alpha + \theta\) |
| 顺时针 | \(\begin{pmatrix} x\cos\theta + y\sin\theta \\ -x\sin\theta + y\cos\theta \end{pmatrix}\) | \(\alpha - \theta\) |

## 4. 几何性质
- **模长不变性**：
  \[
  \|\vec{a}'\| = \|\vec{a}''\| = \|\vec{a}\|
  \]
- **角度关系**：
  \[
  \angle(\vec{a}') = \alpha + \theta,\quad \angle(\vec{a}'') = \alpha - \theta
  \]

## 5. 示例验证
令 \(\vec{a} = \begin{pmatrix} 1 \\ 0 \end{pmatrix},\ \theta = \frac{\pi}{2}\)：
\[
\begin{aligned}
\text{逆时针：} & \begin{pmatrix} 1\cdot0 - 0\cdot1 \\ 1\cdot1 + 0\cdot0 \end{pmatrix} = \begin{pmatrix} 0 \\ 1 \end{pmatrix} \\
\text{顺时针：} & \begin{pmatrix} 1\cdot0 + 0\cdot1 \\ -1\cdot1 + 0\cdot0 \end{pmatrix} = \begin{pmatrix} 0 \\ -1 \end{pmatrix}
\end{aligned}
\]

## 6. 扩展公式

### 6.1 复数表示法
\[
(x + yi)e^{i\theta} = \underbrace{(x\cos\theta - y\sin\theta)}_{\text{实部} } + i\underbrace{(x\sin\theta + y\cos\theta)}_{\text{虚部} }
\]

### 6.2 三维旋转（罗德里格斯公式）
对于旋转轴单位向量 \(\hat{k}\)：
\[
\vec{v}' = \vec{v}\cos\theta + (\hat{k}\times\vec{v})\sin\theta + \hat{k}(\hat{k}\cdot\vec{v})(1-\cos\theta)
\]

## 7. 计算步骤
1. 计算原始角度 \(\alpha = \arctan\left(\frac{y}{x}\right)\)
2. 计算新角度：
   - 逆时针：\(\alpha_{\text{new} } = \alpha + \theta\)
   - 顺时针：\(\alpha_{\text{new} } = \alpha - \theta\)
3. 转换为直角坐标：
   \[
   \vec{a}_{\text{rotated} } = r\begin{pmatrix} \cos\alpha_{\text{new} } \\ \sin\alpha_{\text{new} } \end{pmatrix}
   \]

## 8. 矩阵乘法验证
通过展开矩阵乘法验证公式正确性：
\[
\begin{pmatrix} a & b \\ c & d \end{pmatrix}\begin{pmatrix} x \\ y \end{pmatrix} = \begin{pmatrix} ax + by \\ cx + dy \end{pmatrix}
\]

**逆时针旋转验证**：
\[
\begin{pmatrix} \cos\theta & -\sin\theta \\ \sin\theta & \cos\theta \end{pmatrix}\begin{pmatrix} x \\ y \end{pmatrix} = \begin{pmatrix} x\cos\theta - y\sin\theta \\ x\sin\theta + y\cos\theta \end{pmatrix}
\] 

### 点线最近距离
在二维平面上有两个不同的点 $A(x_A,y_A)$ 和 $B(x_B,y_B)$ 构成直线/线段 $AB$ ，你需要构建一个数据结构，使得其能支持：  
- 点-直线最近点：输出点 $C(x_c,y_c)$ 到直线 $AB$ 的最近点；  
- 点-直线最近距离：输出点 $C$ 到直线 $AB$ 的最近距离；  
- 点-线段最近点：输出点 $C$ 到线段 $AB$ 的最近点；  
- 点-线段最近距离：输出点 $C$ 到线段 $AB$ 的最近距离。
```c++
#include <bits/stdc++.h>
const int N=2e5+10;
const int inf = 0x3f3f3f3f;
using namespace std;
using ull = unsigned long long int;
using ll = long long int;
using ld = long double;
struct point {
    ld x,y;
    point operator+(const point &a)const{
        return {x+a.x,y+a.y};
    }
    point operator-(const point &a)const{
        return {x-a.x,y-a.y};
    }
    point operator*(const ld &a)const{
        return {x*a,y*a};
    }
    point operator/(const ld &a)const{
        return {x/a,y/a};
    }
    ld operator^(const point &a)const{
        return x*a.x+y*a.y;
    }
    ld crs(const point &a)const{
        return x*a.y-y*a.x;
    }
};
ld disf(point const &A,point const &B){
    return (A.x-B.x)*(A.x-B.x)+(A.y-B.y)*(A.y-B.y);
}
ld dis(point const &A,point const &B){
    return sqrt((A.x-B.x)*(A.x-B.x)+(A.y-B.y)*(A.y-B.y));
}
int main() {
    ios::sync_with_stdio(false);
    cin.tie(0);
    int _;
    cin>>_;
    while(_--){
        int op;
        point A,B,C;
        cin>>op>>A.x>>A.y>>B.x>>B.y>>C.x>>C.y;
        point AC=(C-A);
        point AB=(B-A);
        ld h=AB.crs(AC)/dis(A,B);
        point v={-AB.y,AB.x};
        point X=C-(v*h/dis(A,B));
        cout<<fixed<<setprecision(12);
        if(op==1){
            cout<<X.x<<" "<<X.y<<'\n';
        }
        else if(op==2){
            cout<<abs(h)<<'\n';
        }
        else if(op==3){
            if(X.x<=max(A.x,B.x)&&X.x>=min(A.x,B.x)&&X.y<=max(A.y,B.y)&&X.y>=min(A.y,B.y)){
                cout<<X.x<<" "<<X.y<<'\n';
            }
            else{
                if(disf(A,C)<=disf(B,C)){
                    cout<<A.x<<" "<<A.y<<'\n';
                }
                else{
                    cout<<B.x<<" "<<B.y<<'\n';
                }
            }
        }
        else if(op==4){
            if(X.x<=max(A.x,B.x)&&X.x>=min(A.x,B.x)&&X.y<=max(A.y,B.y)&&X.y>=min(A.y,B.y)){
                cout<<abs(h)<<'\n';
            }
            else{
                if(disf(A,C)<=disf(B,C)){
                    cout<<dis(A,C)<<'\n';
                }
                else{
                    cout<<dis(B,C)<<'\n';
                }
            }
        }
    }
    return 0;
}
```
